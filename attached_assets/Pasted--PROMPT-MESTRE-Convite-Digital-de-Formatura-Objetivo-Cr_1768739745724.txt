🎯 PROMPT MESTRE — Convite Digital de Formatura
Objetivo: Criar um convite digital de formatura elegante e moderno em React, com foco total na estrutura visual, hierarquia de componentes e interatividade. Pronto para produção.

📦 Stack Tecnológica
Categoria	Tecnologia
Framework	React 18+
Linguagem	TypeScript
Build Tool	Vite
Estilização	Tailwind CSS 3.4+
Componentes UI	shadcn/ui (baseado em Radix UI)
Animações	Framer Motion
Ícones	Lucide React
Carousel	Embla Carousel (via shadcn/ui)
Fontes	Google Fonts: Cormorant Garamond, Montserrat, Great Vibes
Routing	React Router DOM v6
🎨 Paleta de Cores (Tema Formatura Clássica)
Paleta elegante inspirada em cerimônias acadêmicas: Azul Marinho, Dourado e Bordô.
